# Aura AI 1.2.0 — Native macOS Harness

Aura AI 1.2.0 is a full macOS revamp. The supported product is now a native SwiftUI agent harness with explicit tool permissions, real document generation, inspectable Markdown memory, local-model support, and complete English and Korean editions.

## Highlights

- Permissioned workspace, file-write, shell, and macOS-control tools
- Markdown, HTML, Excel, Word, and PowerPoint skills
- Document extraction, scanned-PDF OCR, and vision attachments
- Verified generated-file attachments with split preview
- Post-response memory curator using rolling conversation and attachment context
- Korean UI, characters, model responses, and stored memories
- Local llama.cpp plus OpenAI, Anthropic, Gemini, Grok, and compatible providers
- Local cloud privacy review before sensitive text leaves the Mac

# Aura AI 1.2.0 — 네이티브 macOS 하니스

Aura AI 1.2.0은 macOS용 전체 개편 릴리즈입니다. 지원 제품은 이제 명확한 도구 권한, 실제 문서 생성, 확인 가능한 Markdown 기억, 로컬 모델 연결, 완전한 영어 및 한국어 에디션을 갖춘 네이티브 SwiftUI 에이전트 하니스입니다.

## 주요 변경 사항

- 권한이 제한된 작업 폴더, 파일 작성, 셸, macOS 제어 도구
- Markdown, HTML, Excel, Word, PowerPoint 기술
- 문서 추출, 스캔 PDF OCR, 이미지 비전 첨부
- 결과 검증을 거친 생성 파일과 분할 미리보기
- 롤링 대화와 첨부 맥락을 사용하는 응답 후 기억 정리 에이전트
- 한국어 UI, 캐릭터, 모델 응답, 저장 기억
- 로컬 llama.cpp 및 OpenAI, Anthropic, Gemini, Grok, 호환 제공자
- 민감한 텍스트를 전송하기 전 로컬 클라우드 개인정보 검토

Apple silicon 및 macOS 15 이상이 필요합니다. 커뮤니티 배포 파일은 임시 서명되며 Apple 공증은 받지 않았습니다.
